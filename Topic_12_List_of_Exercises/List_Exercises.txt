Udemy Course - Introduction to the Python Language
Instructor: Dr. Diego Mariano

List of Exercises 

	1. Make a program that takes the age of the user and say if he is of legal age or minor.

	2. Make a program that receives two notes typed in by the user. If the grade is greater than or equal to six, write approved, otherwise write disapproved.

	3. Write a program that solves a quadratic equation.

	4. Write a program that sorts a numeric list with three elements.

	5. Write a program that takes two numbers and a sign, and perform the math operation defined by the sign.
